# ⚡ Fluke XD Shop — Premium AI Reseller & Lease Platform

**Fluke XD Shop** คือเว็บไซต์ร้านค้านำเสนอและแนะนำสิทธิ์การใช้งานบัญชี AI พรีเมียมระดับโลก (Premium AI Accounts & Licensing Reseller) ที่ได้รับการออกแบบและพัฒนาด้วยดีไซน์ระดับโมเดิร์นหรูหรา (Deep Ocean Premium Theme) เพื่อให้ผู้ใช้งานสามารถเข้าถึงระบบผู้ช่วยปัญญาประดิษฐ์ระดับแนวหน้าได้อย่างสะดวก รวดเร็ว ปลอดภัย และเสถียรที่สุด

---

## 🎯 วัตถุประสงค์ของโปรเจกต์ (Project Purpose)
เว็บไซต์นี้จัดสร้างขึ้นเพื่อเป็นช่องทางหน้าร้านหลักในการนำเสนอแพ็กเกจการใช้งานบัญชี AI ยอดนิยม เช่น **ChatGPT Plus**, **Gemini Pro (Google AI Pro)**, และ **Super GROK** โดยมีรายละเอียดเงื่อนไขและราคาที่ชัดเจน เพื่อเป็นตัวกลางในการคัดสรร แนะนำ และจำลองสิทธิ์การเข้าใช้งาน พร้อมระบบสนับสนุนลูกค้าหลังการขายในช่วงเวลาทำการ

---

## 💻 ประเภทเว็บไซต์ (Web Application Category)
- **Category**: E-Commerce & Product Showcase Landing Page (Single Page Application)
- **Theme**: Deep Ocean Premium (Midnight Blue & Soft Blue Gradients with Glassmorphism)
- **Target Audience**: กลุ่มนักพัฒนา โปรแกรมเมอร์ ดีไซเนอร์ นักเรียน นักศึกษา และบุคคลทั่วไปที่ต้องการใช้งานบัญชี AI ระดับโปรในราคาประหยัดและมีการรับประกันระบบ

---

## ✨ คุณลักษณะเด่นของระบบ (Core Features)

1. **Multilingual Switcher (TH/EN)**: ระบบรองรับสองภาษาอย่างสมบูรณ์แบบเพื่อความสากลในการเข้าถึงข้อมูล
2. **Detailed Product Modals**: แสดงข้อมูลรายละเอียดการใช้งานแพ็กเกจอย่างละเอียด:
   * **ChatGPT Plus**: รายละเอียดการแชร์อีเมลร้าน (4 คน) เทียบกับแบบบัญชีส่วนตัว และการโอนย้ายข้อมูล
   * **Gemini Pro**: สิทธิ์การแชร์พื้นที่ครอบครัว 5 คน เงื่อนไขหัวแฟมิลี่ และวิธีการเข้ากลุ่ม
   * **Super GROK**: ข้อมูลเวลาทำการและเงื่อนไขการสนับสนุนลูกค้า
3. **Interactive Dropdowns**: ดรอปดาวน์คลาสซีกลาสโซฟิสติเคทบนเดสก์ท็อป และระบบพับ/คลี่รายการ (Collapsible Drawer Menu) บนสมาร์ทโฟน
4. **Perfect Mobile Responsiveness**: แสดงผลสมบูรณ์แบบบนหน้าจอสัมผัสทุกขนาด ทั้งระบบปฏิบัติการ **iOS (Safari)** และ **Android (Chrome)**
5. **Direct Contact Integration**: ลิงก์ตรงเพื่อแชทสั่งซื้อทาง LINE OA และ Facebook พร้อมรูป QR Code ที่สแกนง่าย

## 🛠️ เทคโนโลยีที่เลือกใช้ (Technology Stack)

- **Frontend Core**: React 19
- **Build System**: Vite 6
- **Styling**: Tailwind CSS v4 (พร้อมใช้งาน Utility class และตัวแปร CSS Variables ในระดับสูงสุด)
- **Icons**: Lucide React
- **Aesthetic Accents**: Glassmorphism, radial gradient blobs, webkit blur filters, and micro-interactions

## 🚀 วิธีการติดตั้งและรันโปรเจกต์ (Installation & Running Guide)

### 1. ติดตั้ง Dependencies
```bash
npm install
```

### 2. รันโปรเจกต์ในโหมดพัฒนา (Development Mode)
```bash
npm run dev
```
เปิดใช้งานผ่านเบราว์เซอร์ที่ลิงก์: `http://localhost:5173`

### 3. ตรวจสอบการคอมไพล์และ Build Production
```bash
npm run build
```
